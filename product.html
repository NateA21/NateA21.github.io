Instructions for use:

Logging in:

Login with a current Particle account, or simply enter a valid email and password and click create new account. This will create a new account for you on Particle. Afterwards, click Login again in order to ensure your account has been created. The Login function will also create a new user for you to act as on the Proficloud servers.

Manage Device:

This screen, combined with the Device ITTT page, will allow you to control your device while connected via serial connection. Simply enter your port into the field below the Connect Serial button and then click said button. A response will show in the field above. You can disable and enable channels from this page as well. (Note: This page will be combined with Device ITTT before release)

Device ITTT:

This page will allow you to code your own basic logic if statements and send them down to the device. If you are unable to connect, head back to the Manage Device page and try to connect via serial again. To use the blocks, click on the tabs on the left and drag them out. A simple command would be something like :"Set digitalInput1 on". A simple if statement would be "if digitalInput1 > digitalInput2, set digitalInput1 off" (Note: This page will be combined with Manage Device before release)

Debug:

The debug page will allow you to send commands and receive data through Particle's cloud server, with your device. Your device must be connected via Wifi in order for this page to work. If you are connected with a valid token, click the List Devices button to see all devices your account has. You can then unclaim devices, claim devices, or set a device to active in order to disable and enable channels.

Cloud Services

The cloud services page will allow you to use proficloud's server in order to list devices, add devices, delete devices, and more. (These features are not fully implemented yet!)
